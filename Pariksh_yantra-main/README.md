# Parikshayanta

An exam management system for creating and conducting online tests.

## Features
- Teacher portal for exam creation
- Student portal for taking tests
- Question database management
- Test evaluation system

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Run the application

## Structure
- `teacher.py` - Teacher portal functionality
- `student.py` - Student portal functionality
- `Database/` - Question banks and test data
- `UI/` - HTML templates and static files
